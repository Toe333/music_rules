# `data/` — rule corpora and source materials

This folder is **reference material**, not part of the installed Python package.
The runtime-canonical rule corpus lives at
`src/music_rules/data/rules_combined.json` so that `importlib.resources` can
load it from an installed wheel without any filesystem assumptions.

## What's in here

| Path | What it is | License / provenance |
|---|---|---|
| `eis/EIS_MASTER_RULES.md` | Human-readable EIS rulebook, 18 sections, provenance-tagged `[TG]` (Ted Greene), `[Web]`, `[Both]`. | Distilled / paraphrased from Spud Murphy's EIS as taught by Ted Greene (1977-78 lesson PDFs) + public web sources. Short quotations (<15 words) attributed inline. |
| `eis/extracted/` | Raw OCR / text extractions from the 5 Ted Greene EIS PDFs and one web supplement. **Development reference only** — excluded from the wheel via `pyproject.toml`. | Ted Greene estate; not redistributed. The PDFs themselves stay outside this repo. |
| `eis/extracted/EIS_web_supplement.md` | Web-sourced supplementary EIS material. | Mixed; treat any future Stack Exchange additions as CC BY-SA 4.0 and attribute. |
| `fux/Counterpoint_rules_table.csv` | Spreadsheet view of the 39 Fux rules. | Original to this project. |
| `fux/FuxCP5_attribution.md` | Acknowledgment of the FuxCP5 reference implementation. | Documentation. |
| `tables/EIS_rules_table.{xlsx,csv}` | Spreadsheet view of the 119 EIS rules. | Original to this project. |
| `tables/Counterpoint_rules_table.xlsx` | Spreadsheet view of the 39 Fux rules. | Original to this project. |
| `schema/rules.schema.json` | JSON Schema for `rules_combined.json` (generated by `scripts/`). | Original to this project. |

## Source-of-truth rule

There is **one** canonical rule file:

```
src/music_rules/data/rules_combined.json
```

The CSV / XLSX views in `tables/` and `fux/` are *exports*, regenerable from
the JSON via `scripts/export_tables.py` (Phase 7). If you ever find yourself
hand-editing a CSV, stop — edit the JSON, then re-run the exporter.
